### 📂 **𝐅𝐈𝐋𝐄 𝐒𝐓𝐑𝐔𝐂𝐓𝐔𝐑𝐄**
```text
TOM-PRIME-X-MINI/
├── commands/           # All bot command modules
│   ├── admin/          # Admin-only commands (Anti-link, Kicking)
│   ├── general/        # User commands (Menu, Ping)
│   └── tools/          # Utility tools (Sticker maker, Logo)
├── utils/              # Helper functions & database handlers
├── handler.js          # Core message processing engine
├── config.js           # Main configuration file
├── database.js         # MongoDB/Local JSON connection
└── index.js            # Entry point for the bot
🧩 𝐒𝐄𝐓𝐔𝐏 & 𝐂𝐎𝐍𝐅𝐈𝐆𝐔𝐑𝐀𝐓𝐈𝐎𝐍
​Before deploying, make sure to set the following Environment Variables:
# Mandatory Variables
SESSION_ID   = "YOUR_SESSION_ID"
PREFIX       = "."
